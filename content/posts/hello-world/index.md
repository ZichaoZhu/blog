欢迎来到我的个人博客！这是我的第一篇文章。

## 关于本博客

本博客是基于 **Next.js 14** 构建的现代化个人博客系统，采用了最新的技术栈：

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- MDX
- Framer Motion

## 功能特性

### 1. 精美的动画效果

首页包含了多层动画效果：
- 渐变背景
- 视差滚动
- 打字效果
- 粒子特效

### 2. Markdown 支持

支持完整的 Markdown 语法，包括：

- **粗体文本**
- *斜体文本*
- `代码片段`
- [链接](https://example.com)

### 3. 代码高亮

```typescript
const greeting = (name: string): string => {
  return `Hello, ${name}!`;
};

console.log(greeting('World'));
```

### 4. 列表

有序列表：
1. 第一项
2. 第二项
3. 第三项

无序列表：
- 项目一
- 项目二
- 项目三
